As part of my research focuses on improving the tools used in the study of plasma wakefield acceleration (PWFA) and the simulation of PWFAs and associated scenarios, a significant portion of my time is dedicated to working with software. Included below are tools for analyzing Gaussian beam data, custom code I've written to save time on repetitive tasks, and various other useful resources I've created.

## GaussBean

[![Static Badge](https://img.shields.io/badge/GitHub-gray?logo=github&labelColor=gray)](https://github.com/leahghartman/GaussBean)
[![PyPI - Version](https://img.shields.io/pypi/v/GaussBean?label=PyPI)](https://pypi.org/project/gaussbean/)

_GaussBean_ is a Gaussian beam analysis repository, initially developed to store code used to analyze passive plasma lens data within the CU WARG group. The subsequent package built for PyPI contains functions for processes such as image filtering, cropping, and background subtraction. It also includes convenient plotting functions for frequently-used techniques and supports data analysis for both extensive datasets and individual images.
